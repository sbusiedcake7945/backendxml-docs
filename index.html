The Engine Architecture

The BackendXML ecosystem follows a Tri-Phase Pipeline to ensure maximum safety and speed.

1. Stream Lexing

Raw XML is shredded into logical tokens, identifying attributes, content, and nested children.

2. AST Mapping

The Parser validates the token stream to build an Abstract Syntax Tree, checking for language integrity.

3. Concurrent Exec

The Interpreter traverses the AST, spawning Goroutines for HTTP handlers and background tasks.

Technical API: Elements

Tag Context Capabilities
<backend> Root The global application scope. Parent of all nodes.
<import> Global Imports .backend files or recursively loads entire directories.
<function> Handler Native code gateways. Executes raw Go, JS, or HTML.
<var> Hybrid Dual-scope variables (Global/Request). Supports {{var}} injection.
<route> Global Static and Dynamic HTTP path routing. Thread-safe execution.
<database> Global Unified data gateway. Supports SQLite and JSON storage.
<if> Handler Conditional execution using JS-style expressions. Supports <elseif> and <else>.
<get> Hybrid Injects request metadata (ip, url). Security-hardened against DB injection.
<include> Global C-style inline file inclusion with detailed includeError tracing.
<open> Global Asynchronous file execution in an isolated goroutine.
<input> Handler Waits the request until the console accepts it
<call> / Custom Handler Executes named functions. Supports isolated parameter passing.

Architecture: Modularity

Maintain clean code with the Recursive Importer. Perfect for large-scale microservices.

<import path="modules/auth" />
Automatically finds and mounts all .backend logic files within the directory.

Config: The Core Shell

Fine-tune your instance using the <config> directive.

<config>
  <port>443</port>
  <start_message>7945 Core Engine Online at {{port}}</start_message>
</config>

Unified Data Layer

One tag to rule them all. Switch between SQL and JSON with a single attribute.

SQLite (Default)

<database connection="./data.db" />

JSON Storage

<database type="json" connection="./data.json" var="db" />
Pro Tip: All database paths support ./ relative syntax for maximum portability.

Conditionals: Pure Intelligence

Control the flow of your application with high-level logic blocks.

<if condition="'{{user_id}}' == '7945'">
  <print>Welcome, Developer!</print>
<elseif condition="{{age}} >= 18">
  <print>Access Granted.</print>
<else>
  <print>Access Denied.</print>
</if>

Metadata: Environmental Awareness

Inject real-time request data directly into your views or logic.

Available data points:

  • <get data="ip" /> - Resolves to the client's IP address.
  • <get data="url" /> - Resolves to the full requested path.
Security Alert: Metadata tags are automatically sanitized and blocked within <query> blocks to prevent data leaks.

Modularity: Scale With Elegance

Split your application into manageable components without losing performance.

<include file="..." />

Seamlessly merges external logic into the current scope. If an error occurs, the engine provides an includeError trace pointing to the exact line in the included file.

<open file="..." />

Launches a file in a completely isolated background thread. Ideal for audit logging, notification dispatching, or cleanup tasks.

Custom Tags

Elevate your code by using named functions as native XML elements.

<!-- Definition -->
<function name="alert" type="html">
  <div class="alert-{{type}}">{{message}}</div>
</function>

<!-- Usage -->
<alert type="error" message="Critical Failure!" />

Advanced Mastery: Encryption

Handling secure data is a first-class citizen in BackendXML using Go native blocks.

<route path="/encrypt">
  <handler>
    <var name="secret" value="7945_CODE_PLATINUM" />
    <function type="go">
      // Example: Base64 Obfuscation Logic
      encoded := base64.StdEncoding.EncodeToString([]byte("{{secret}}"))
      fmt.Fprint(w, encoded)
    </function>
  </handler>
</route>
Developer Tip: Use the <function type="go"> block to access the full crypto/ package from the Go standard library for AES or RSA implementations.
